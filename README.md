# Cloud-Based Smart Attendance & Employee Management System

## Project Description
This project is a simple web application developed using HTML and CSS. It helps organizations manage employee information and attendance through a user-friendly interface.

## Features
- Employee Login
- Attendance Statistics
- Attendance Records
- Employee Details
- Contact Section

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download the project files.
2. Open the project folder.
3. Double-click `index.html` to open the website in a web browser.

- index.html
- style.css
